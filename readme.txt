E-mail address You will find on <a href="http://blog.meloniq.net/kontakt/">contact page</a>.



== Installation ==

1. Upload the folder 'new-twitter-widget' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Appearance->Widgets" menu and add twitter widget to sidebar.
