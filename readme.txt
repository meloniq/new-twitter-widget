Shows the latest tweets from a Twitter account in a sidebar widget. 

== Description ==

This plugin displays the latest posts from a Twitter account in a sidebar widget.

= Features / Highlights =
* Widget title
* Number of tweets to show
* Show/Hide tweet date
* Uses cache

= No need to create Twitter Application and enter keys - uses <a href="http://twitter-rss.com">Twitter RSS</a> service! =


= Supported languages =
* English
* Polish

If You translated plugin to Your native language, please send it to me, and will attach it to next release.
E-mail address You will find on <a href="http://blog.meloniq.net/kontakt/">contact page</a>.



== Installation ==

1. Upload the folder 'new-twitter-widget' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Appearance->Widgets" menu and add twitter widget to sidebar.
